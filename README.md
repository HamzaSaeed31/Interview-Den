# FYP - Interview Den

Final Year Project completed over two semesters at FAST-NUCES (2024-2025).

## Structure

- **fyp-1/** - Semester 8: Proposal, mid-evaluation and defense materials
- **fyp-2/** - Semester 9: Full implementation
  - `FYP_AI_Components-master/` - AI/ML backend (Groq, resume matching, quiz generation)
  - `InterviewDen-main/` - Next.js + Supabase web application
  - `Final-eval/` - Final evaluation submission
  - `Mid-eval/` - Mid-evaluation submission

## Description

Interview Den is an AI-powered mock interview platform:
- Resume screening and AI-based job matching
- Auto-generated technical quiz assessments
- Video interview simulation with AI avatar
- Company and candidate dashboards (Supabase backend)

> **Note:** Groq API keys have been redacted. Set them via environment variables.
